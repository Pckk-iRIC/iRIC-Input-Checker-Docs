インストール¶

動作前提¶

  • OS: Windows 10 / 11
  • iRIC 実行環境が利用可能であること

導入手順¶

  1. リリースページから配布バイナリ（zip）をダウンロードします。
  2. 任意のフォルダへ展開します。
  3. 展開先の実行ファイルを起動します。

初回起動時の確認¶

  • 起動時にエラーが表示される場合は、トラブルシュート を参照してください。
