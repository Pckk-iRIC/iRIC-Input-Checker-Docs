トラブルシュート¶

起動に失敗する場合¶

  • iRIC 実行環境が導入済みであることを確認してください。
  • 配布物を別フォルダへ再展開し、再実行してください。

入力エラーが表示される場合¶

  • 入力に .ipro または project.xml を指定していることを確認してください。
  • project.xml を指定する場合、同じフォルダ内にプロジェクトファイル一式が存在することを確認してください。

保存先エラーが表示される場合¶

  • 保存先フォルダに入力プロジェクト配下を指定することはできません。
  • 入力ディレクトリとは別の出力ルートを指定してください。

PDF レポートが生成されない場合¶

  • HTML レポートが生成されているかを確認してください。
  • 画像出力件数を減らして再実行してください。

HTML レポートで画像や詳細確認が開けない場合¶

  • GUI の 簡易レポート（HTML）を開く から開いているかを確認してください。
  • 出力フォルダ内の web/report/index.html, web/detail/index.html, plots/, data/ をまとめて保持しているかを確認してください。
  • report または detail の HTML だけを単独で別フォルダへ移動すると、画像や詳細確認へのリンクが切れることがあります。

詳細確認 viewer が重い場合¶

  • まず簡易レポートで概要を確認し、必要な属性だけ詳細確認してください。
  • 比較モードでは、差分図は初期状態で閉じています。必要な項目だけ 差分図を表示 を開いてください。
  • 他のアプリケーションがメモリを多く使用している場合は、不要なアプリを閉じてから再度開いてください。

文字化けが発生する場合¶

  • 元データの文字コードを確認してください。
  • 表示側アプリケーションの文字コード設定を確認してください。
